# 🎮 Tic Tac Arena

A simple Tic Tac Toe game built with HTML, CSS, and JavaScript.

## Features
- Player vs Player
- Player vs AI (Easy, Medium, Hard)
- Dark/Light Mode
- Score Board
- Restart & Reset
- Responsive Design

## Files
- index.html
- game.html
- style.css
- script.js
- game.js

## Technologies
HTML • CSS • JavaScript
