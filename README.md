## ✨ Features
- ❌ Player vs Player
- 🤖 Player vs AI (Easy, Medium, Hard)
- 🌙 Dark / Light Mode
- 📊 Live Score Board
- 🔄 Restart Match
- 🗑️ Reset Scores
- 📱 Responsive Design
